# 🚀 My Portfolio

A modern, interactive portfolio website showcasing my skills, projects, and professional journey.

</div>

## ✨ Features

- **Interactive UI** — Smooth animations and transitions using Framer Motion
- **AI-Powered Chat** — Integrated Gemini AI for intelligent interactions
- **Responsive Design** — Fully optimized for all devices
- **Dark Mode** — Clean, modern dark theme
- **Project Showcase** — Featured projects with live demos
- **Skills Section** — Technical expertise visualization
- **Blog Preview** — Latest articles and updates
- **GitHub Activity** — Real-time contribution graph

## 🛠️ Tech Stack

- **Frontend:** React 19, TypeScript, Vite
- **Styling:** Tailwind CSS 4
- **Animation:** Framer Motion, Motion
- **AI:** Google Gemini API
- **Deployment:** Vercel

## 🚀 Getting Started

```bash
# Clone the repository
git clone https://github.com/mdjtk/portfolio.git

# Install dependencies
npm install

# Create environment file
cp .env.example .env.local

# Add your Gemini API key to .env.local
GEMINI_API_KEY=your_api_key_here

# Start development server
npm run dev
```
